# Lemon Squeezy Product Description

## Product Name
polyglot-kit

## Short Description (for checkout page)
113-locale i18n infrastructure for JavaScript. Zero dependencies. One-time purchase.

## Full Description

Drop-in internationalization with zero runtime dependencies. Everything you need to ship your app in 113 languages — without spending days on setup.

**What's included:**

- **113 locales** with full metadata — label, text direction, currency, numbering system. 5 RTL locales (Arabic, Hebrew, Persian, Urdu).
- **206 pre-translated UI strings** across 11 namespaces: common, status, auth, validation, time, navigation, actions, dialog, accessibility, pagination, empty.
- **Formatting utilities** — currency, dates, and numbers using the native Intl API. Handles zero-decimal currencies (JPY, KRW), Indian number grouping, relative time, and more.
- **Framework adapters** — drop-in support for next-intl, react-intl, i18next, and vanilla JS. The core is framework-agnostic.
- **TypeScript strict** — full type definitions, `SupportedLocale` union type, type guards for validation.
- **Tree-shakable** — ESM + CJS dual output. Import only locales, formatting, or messages independently.

**How it works:**

1. Purchase and receive a GitHub collaborator invitation (within minutes)
2. Install: `npm install git+https://github.com/TomahawkCM/polyglot-kit.git`
3. Import what you need and ship
